When would you use ‘=’ vs ‘==’ vs ‘.equals’


The '=' is the assignment operater and allows us to assign a new value
to a variable. Its very easy to make the common mistake of using the '='
assignment operater while testing a condition such as in a if( 5 = 3)
.. if you do this your code will not compile!

The '==' is the equality operater. This operator is used to compare
primitive types like integers. So if you use the '==' equality operater
to compare two strings the result may not be correct. The reason for this
is that a string is an 'object' and an object is a reference type.
A reference type does not contain the data like a primitive type such
as an integer so this is when you would use the string method '.equals'.
