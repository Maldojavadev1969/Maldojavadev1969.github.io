What is ‘void’ in a method declaration?


What is ‘void’ in a method declaration? How does this differ from declaring a
method with a return type?

Void in a a method declaration lets the compiler know that there will be nothing
returned back to the line of code that called the method.
When you declare a method with a return type the compiler will know what the
type should be and to check for that result type prior to returning that value to
the calling line of code.
