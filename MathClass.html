‘Math.random()’


To generate a random number you use ‘Math.random()’. What is ‘Math’? What is the
return type of the ‘.random()’ method. Why is the ‘M’ in ‘Math’ capitalized
Math is a Java class that has dozens of math methods that can be utilized such as
Math.random() method. The return type of .random is a random floating
point number between 0 and 1.
The reason that the 'M' in Math is capitalized is to distinguish class names
from member names.
The random method is found in the following package: java.lang.Math.
